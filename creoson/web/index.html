Welcome to OpenSource Automation!

CREOSON, the first OpenSource Automation Tool for PTC's CREO Parametric!

CREOSON is a highly optimized and simplified interface to CREO's JLINK Library that can be called using simple JSON Requests from ANY Language over an HTTP (Web Protocol) Interface.
